class gama.utilities.generic.function_dispatcher.FunctionDispatcher(n_jobs, func)[source]¶

Bases: object

A manager for evaluating functions async in the background using multi-processing.

This object will spawn n_jobs child processes which will run func on any input given through queue_evaluation. Return values of evaluations can be obtained by calling get_next_result. To keep track of which input leads to which output, queue_evaluation returns a unique identifier for each call. Finally, get_next_result will return the identifier and item alongside the output of func(item).

get_next_result()[source]¶

Get the result of an evaluation that was queued by calling queue_evaluation. This function is blocking.

This function raises a ValueError if it is called when there is no job queued with queue_evaluation.
